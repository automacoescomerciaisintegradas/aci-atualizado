# Wordpress Social Strategy Workflow

Este workflow automatiza a coleta e curadoria de notícias de WordPress para redes sociais.

1. **Acesso às Fontes**: Acesse as seguintes URLs:
   - https://www.painelwp.com.br/newsletter-wordpress/
   - https://wordpress.org/news/
   - https://shopee.com.br/blog/news/
   - https://shopee.com.br/news/

2. **Análise de Conteúdo**:
   - Identifique tópicos relevantes para desenvolvedores PHP/WordPress.
   - Selecione os 5 itens com maior potencial de engajamento.

3. **Sugestão de Posts**:
   - Crie 5 sugestões de posts curtos (estilo Twitter/LinkedIn).
   - Adicione o link da fonte original.

4. **Armazenamento**:
   - Salve em `social-media/ideias/ideias-semana-[WW]-[YYYY].md`.

5. **Finalização**: 
   - Notifique o usuário sobre o local do arquivo.
